# 剩余金额计算器

一个简单实用的网页版剩余金额计算工具，帮助用户快速计算服务或产品的剩余价值。

## 功能特点

- 支持多种时间单位计算（年、季度、月、184天）
- 实时显示当前日期和时间
- 简洁直观的用户界面
- 详细的计算结果展示
- 支持键盘快捷操作
- 响应式设计，适配各种设备

## 使用说明

1. 选择开通日期
   - 点击日期输入框选择服务开始日期
   - 日期不能超过当前日期

2. 选择时间单位
   - 年（365天）
   - 季度（90天）
   - 月（30天）
   - 184天

3. 输入原始价格
   - 输入需要计算的服务或产品原价
   - 金额必须大于0

4. 查看计算结果
   - 点击"计算"按钮或按回车键进行计算
   - 结果将显示详细信息，包括：
     * 开通日期
     * 当前日期
     * 已使用天数
     * 剩余天数
     * 原价
     * 剩余金额

## 快捷操作

- 按回车键：快速计算
- 点击任意键：关闭结果弹窗
- 点击刷新按钮：重置所有输入

## 技术实现

- 纯前端实现，无需后端支持
- 使用HTML5、CSS3和原生JavaScript
- 响应式设计，支持移动端访问
- 精确的时间计算逻辑

## 使用方法

直接在浏览器中打开`calculator.html`文件即可使用。

## 注意事项

- 所有时间计算均基于本地时间
- 计算结果精确到小数点后两位
- 建议使用现代浏览器以获得最佳体验
